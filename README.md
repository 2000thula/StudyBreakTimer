Study Break Timer App
Project Description

The Study Break Timer App is an Android mobile application developed to help students manage their study time effectively. The application allows users to track study sessions, take scheduled breaks, and review their past study activity.

Features
User Registration
User Login & Logout
Study Timer
Break Timer
Session History Tracking

Technologies Used
Android Studio
Java
SQLite Database
GitHub for Version Control

Database Tables
1. Users Table
 id
username
password

2. Sessions Table
id
study_time
break_time
date

How the App Works
1. User registers a new account.
2. User logs into the system.
3. User starts a study timer.
4. User can take breaks during study.
5. Study session details are saved to the database.
6. User can view session history.
